# dental-implant-fatigue-analysis
Python tools for dental implant fatigue analysis according to ISO 14801.
Biomedical Engineering project developed during the Master's Degree in Clinical Engineering.

## Objectives

- Fatigue testing according to ISO 14801
- Failure analysis
- Regression modelling
- Data visualization

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib

## Project Structure

scripts/ -> analysis code

dataset/ -> technical data

results/ -> output reports

images/ -> figures and plots
